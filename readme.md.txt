## Description

Desktop automation for Gradestar application using WinAppDriver

## Technologies used
Selenium for WindowsDriver, testNG for test management, Maven for dependencies management, extent reporting, retry anlalyzer

### pre-requisites

#1. make sure jdk, maven is installed in the system
#2. make sure SDK kits for appium is installed for capturing the locators
#3. make sure winappdriver is installed in the system. if not download and install from internet ..you can follow this page 
https://github.com/microsoft/WinAppDriver for more details...
#4. make sure developer option is enabled in the system
